# Nexe Chatbot 🤖

Hey there! Welcome to **Nexe Chatbot**. This is a super simple, self-contained AI chat interface I put together that runs entirely in your browser.

It's designed to be clean, fast, and easy to use—without any complicated setup, heavy frameworks, or confusing dependencies.

## What's inside?

- **Just one file:** Everything (HTML, CSS, and vanilla JavaScript) lives perfectly inside `index.html`.
- **Bring your own key:** You just paste your OpenAI (or OpenRouter) API key directly into the UI. Don't worry, it stays right in your browser's local storage and is never sent anywhere else.
- **Auto-detection:** If you paste an OpenRouter key (`sk-or-...`), the code automatically detects it and switches endpoints behind the scenes!
- **Error handling:** Built-in safeguards for when you run out of credits, hit a rate limit, or have network issues.

## How to use it

1. Download or clone this repo.
2. Double click `index.html` to open it in any web browser.
3. Paste your API key in the top right corner and hit "Save Key".
4. Say hi!

*Note: Make sure `logo.png` and `favicon.png` stay in the same folder so the branding shows up properly.*

Feel free to tweak the code, change the styles, and make it your own. Enjoy! 🚀
