# Contributing to Nexe Chatbot

First off, thanks for taking the time to contribute! 🎉

The following is a set of guidelines for contributing to **Nexe Chatbot**. These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create an issue on GitHub. Include:
- A clear title and description.
- Steps to reproduce the bug.
- What you expected to happen vs what actually happened.
- Any relevant screenshots or error messages from the console.

### Suggesting Enhancements
Suggestions are welcome! Please open an issue to discuss your ideas before implementing them.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the project still runs by opening `index.html` in your browser.
4. Submit a pull request!

## Style Guide
- Keep it simple. This project is meant to be a "single file" (mostly) demonstration.
- Use vanilla JavaScript (ES6+).
- Follow the existing CSS structure for styling.

Thank you!
